### 주요 GIT 명령어 ###

```
# 로컬로 Clone (내려받기)
git clone https://github.com/oratune/phmuse.git [로컬디렉토리]

# 연결된 원격 저장소 확인
git remote --verbose

# 신규 파일을 버전관리에 추가
git add <filename>

# 전부다 추가하려면
git add . 
git add --all 

# 수정사항 Commit
git commit -m "수정내용 요약"

# 원격 저장소로 보내기

git push <remote_name> <branch_name>

```
